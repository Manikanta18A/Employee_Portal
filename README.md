# Employee Portal System

## Project Overview
Employee Portal System is a full-stack web application developed to manage employee information, employee records, and organizational data efficiently.

The application provides functionalities for employee management, record maintenance, and employee data handling through a user-friendly web interface.

This project is designed using layered architecture and follows MVC (Model View Controller) design principles for better maintainability and scalability.

---

# Objective
The main objective of this project is to simplify employee data management within an organization by digitizing employee record handling and reducing manual work.

The system helps:
- Store employee details
- Manage employee records
- Update employee information
- Organize employee data efficiently
- Improve employee management operations

- Repository Layer
- DAO Layer
- Entity Layer

The MVC architecture is implemented for proper separation of concerns and maintainable code structure.
